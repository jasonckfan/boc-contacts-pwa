# 清除緩存說明

## 為什麼需要清除緩存？

瀏覽器和 PWA 會緩存舊的數據，導致顯示的是 demo 數據（30人）而不是真實數據（409人）。

## 清除緩存步驟

### 方法 1：強制刷新（推薦）

1. 在 iPhone Safari 中訪問：https://jasonckfan.github.io/boc-contacts-pwa/
2. 點擊地址欄
3. 點擊刷新按鈕並**長按**
4. 選擇「請求桌面網站」然後再選回「請求移動網站」
5. 或者：關閉 Safari 標籤頁，重新打開

### 方法 2：清除 Safari 緩存

1. 打開 iPhone **設置**
2. 滾動到 **Safari**
3. 點擊 **清除歷史記錄與網站數據**
4. 確認清除
5. 重新訪問應用

### 方法 3：重新安裝 PWA

1. 長按主屏幕上的應用圖標
2. 選擇「刪除 App」
3. 在 Safari 中重新訪問：https://jasonckfan.github.io/boc-contacts-pwa/
4. 點擊分享按鈕 →「添加到主屏幕」

## 如何確認數據已更新？

更新成功後，你應該看到：
- 總聯繫人數：**409人**（不是30人）
- 部門數量：**47個**（不是10個）
- 下拉選單顯示真實部門名稱，如：
  - 借記卡及收單技術方案組 (62人)
  - 個人電子渠道組 (31人)
  - 網絡攻擊防禦 (30人)
  - 等等...
