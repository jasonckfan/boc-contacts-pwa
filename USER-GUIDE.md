# 聯繫人管理 PWA - 使用指南

## 快速開始

### 1. 啟動應用

#### 方法一：使用 Node.js (推薦)
```bash
cd /home/jasonckfan/contacts-pwa
npx http-server -p 8080
```
然後在瀏覽器中訪問：`http://localhost:8080`

#### 方法二：使用 Python
```bash
cd /home/jasonckfan/contacts-pwa
python3 -m http.server 8080
```

#### 方法三：使用 VS Code Live Server
在 VS Code 中安裝 Live Server 插件，右鍵點擊 `index.html` 選擇 "Open with Live Server"

### 2. 安裝為 PWA

#### Android (Chrome)
1. 在 Chrome 中打開應用
2. 點擊菜單（三點圖標）
3. 選擇「添加到主屏幕」
4. 確認安裝

#### iOS (Safari)
1. 在 Safari 中打開應用
2. 點擊分享按鈕
3. 選擇「添加到主屏幕」
4. 確認安裝

#### 桌面 (Chrome/Edge)
1. 在瀏覽器中打開應用
2. 點擊地址欄右側的安裝圖標
3. 或點擊菜單 → 「安裝聯繫人管理」

### 3. 使用功能

#### 搜索聯繫人
- 在搜索框輸入姓名、部門、職位或電話號碼
- 實時顯示匹配結果

#### 按部門篩選
- 點擊頂部的部門標籤
- 快速查看特定部門的聯繫人

#### 查看聯繫人詳情
- 點擊任意聯繫人卡片
- 查看完整信息（電話、手機、電郵、備註）
- 點擊電話或郵件圖標直接撥打或發送郵件

#### 添加新聯繫人
- 點擊右下角的「+」按鈕
- 填寫聯繫人信息
- 點擊保存

#### 編輯聯繫人
- 打開聯繫人詳情
- 點擊「編輯」按鈕
- 修改信息後保存

#### 刪除聯繫人
- 打開聯繫人詳情
- 點擊「刪除」按鈕
- 確認刪除

### 4. 離線使用

應用支持完全離線使用：
- 首次加載後，所有數據存儲在本地
- 無網絡時仍可查看、添加、編輯聯繫人
- 頂部會顯示離線狀態指示器

### 5. 數據說明

- 應用包含 30 個示例聯繫人
- 數據存儲在瀏覽器的 IndexedDB 中
- 數據會在瀏覽器會話間保持
- 清除瀏覽器數據會重置為初始數據
